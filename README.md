# Unity Coin Collection Game 🎮

## Overview

A simple 2D Unity-based game where the player collects coins while interacting with basic game mechanics such as movement, collision, and audio feedback.

This project was developed to explore core Unity concepts including scene setup, asset management, and scripting using C#.

---

## Features

* Player movement using keyboard input
* Coin collection system
* Audio feedback on interactions
* Basic game objects and physics handling
* Scene management in Unity

---

## Tech Stack

* Unity (Game Engine)
* C# (Scripting)

---

## Project Structure

* `Assets/` – Game assets, scripts, audio, and materials
* `Packages/` – Unity package dependencies
* `ProjectSettings/` – Unity project configuration

---

## How to Run

1. Clone the repository
2. Open the project in Unity Hub
3. Load the main scene
4. Click **Play** to run the game

---

## Learning Outcomes

* Understanding Unity project structure
* Working with C# scripts in Unity
* Implementing basic game mechanics
* Handling assets like audio and materials

---

## Notes

This is a beginner-level project created for learning purposes and demonstrates fundamental Unity development concepts rather than a full-scale game.
